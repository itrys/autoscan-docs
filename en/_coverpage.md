# AutoScan <small>1.1.0</small>

> Lightweight Spring Boot Starter for Cross-Package Scanning

- 🚀 Auto-scan base packages, configure once, apply globally
- 🎯 Zero configuration for business packages, leveraging @SpringBootApplication default scanning
- 🏗️ Multi-layer infrastructure support, business projects can serve as infrastructure for others
- 🔧 Non-invasive design, no changes to existing code structure
- 📊 Developer-friendly, detailed scanning logs for debugging
- ⚡ Lightweight, no extra dependencies, fully compatible with Spring Boot 3.x/4.x

**v1.1.0 New Features**

- 🌟 Wildcard Package Support - Use `*` and `**` to simplify configuration
- 🚫 Exclude Scanning - Precise control over scanning scope
- 🎨 Custom Annotations - Extend scanning capabilities
